# Простая офлайн-версия сайта 112

Как открыть:
1. Распакуйте архив.
2. Откройте файл `index.html` двойным кликом.
3. Сайт заработает сразу в браузере.

Что есть:
- вход по ФИО и номеру оператора
- тест с подтверждением каждого ответа
- экран результата

Что нет в этой версии:
- Telegram-бота
- базы данных
- серверной части

Для Telegram нужна отдельная серверная версия.
